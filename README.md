Jak uruchomić program:
- uruchom plik server.jar znajdujący się w katalogu AnimalShelterOnlineSystem\out\artifacts\server w wierszu polecenia
  (java -jar server.jar)
- uruchom plik client.jar znajdujący się w katalogu AnimalShelterOnlineSystem\out\artifacts\client w wierszu polecenia
  (java -jar client.jar)
- Aby zobaczyć obsługę wielu próśb oraz kolejkowanie, należy uruchomić jednocześnie min. 3 instancje pliku client.jar
